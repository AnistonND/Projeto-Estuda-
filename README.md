🎓 Plataforma ESTUDA+

📝 Sobre o Projeto

A "Plataforma ESTUDA+" é uma aplicação web interativa desenvolvida para auxiliar estudantes no aprendizado e revisão de conteúdos através de desafios e simulados. O sistema oferece gamificação, ranking de pontuação e recomendações personalizadas, incentivando o usuário a estudar de forma consistente.

O projeto foi desenvolvido com **HTML5, CSS3 e JavaScript puro**, utilizando `localStorage` para persistência dos dados dos usuários, dispensando a necessidade de um backend complexo.

---

🎯 Funcionalidades

🔐 Autenticação
- Cadastro de novos usuários (nome, e-mail e senha)
- Login seguro com validação de credenciais
- Logout automático com remoção da sessão

👤 Painel do Usuário
- Boas-vindas personalizada com nome do usuário
- Exibição de nível atual, pontuação total e medalha (Bronze, Prata, Ouro)
- Barra de progresso para o próximo nível (50 pontos por nível)
- Gráfico de desempenho com histórico dos últimos 7 registros de pontuação


🎮 Modos de Jogo

⚔️ Batalha da Lógica
- Quiz com perguntas de raciocínio lógico e matemática
- Temporizador de 15 segundos por pergunta
- Pontuação: +10 pontos para acertos, +5 pontos para erros
- Feedback imediato com a resposta correta

📝 Simulado
- 15 perguntas de múltipla escolha (matemática, português, história)
- Acúmulo de pontos baseado no desempenho:
  - 15 pontos para 15 acertos
  - 10 pontos para 10–14 acertos
  - 5 pontos para 5–9 acertos
- Explicação detalhada após cada resposta

🏆 Ranking Geral
- Lista de todos os usuários ordenada por pontuação
- Atualização dinâmica a cada interação

💡 Mensagens Inteligentes
- Recomendações de estudo baseadas na pontuação atual do usuário
- Incentivos personalizados conforme o progresso
